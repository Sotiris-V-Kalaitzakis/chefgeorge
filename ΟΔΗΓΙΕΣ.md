# Chef George — Οδηγίες εγκατάστασης με Admin Panel

Το site είναι στατικό, οπότε το admin panel λειτουργεί με **Decap CMS + Netlify**:
συνδέεσαι στο `/admin`, ανεβάζεις φωτογραφίες, και αυτές αποθηκεύονται αυτόματα
στο GitHub repo και δημοσιεύονται από το Netlify. Χωρίς δικό σου server, χωρίς κόστος.

## Δομή αρχείων

```
chef-george-site/
├── index.html          ← η ιστοσελίδα
├── admin/
│   ├── index.html      ← το admin panel (/admin)
│   └── config.yml      ← ρυθμίσεις CMS
├── data/
│   └── gallery.json    ← εδώ αποθηκεύονται οι φωτογραφίες της gallery
└── images/
    └── uploads/        ← εδώ ανεβαίνουν τα αρχεία εικόνων
```

## Βήματα εγκατάστασης (μία φορά, ~10 λεπτά)

1. **GitHub**: Ανέβασε τον φάκελο σε νέο repository (π.χ. `chef-george-site`),
   branch `main`.

2. **Netlify**: New site from Git → διάλεξε το repo → Deploy.
   (Build command: κενό, Publish directory: `/`)

3. **Ενεργοποίηση Identity** (το σύστημα login):
   Netlify Dashboard → Site → **Identity** → *Enable Identity*.

4. **Ενεργοποίηση Git Gateway** (για να γράφει το CMS στο repo):
   Identity → Settings → Services → **Git Gateway** → Enable.

5. **Κλείσε τις ανοιχτές εγγραφές** (να μπαίνεις μόνο εσύ):
   Identity → Settings → Registration → **Invite only**.

6. **Πρόσκληση**: Identity → *Invite users* → βάλε το email του Chef George.
   Θα λάβει email, πατάει τον σύνδεσμο, ορίζει κωδικό — έτοιμος.

## Καθημερινή χρήση

- Πήγαινε στο `το-site-σου.com/admin`
- Σύνδεση με email/κωδικό
- **Gallery — Φωτογραφίες** → *Add Φωτογραφίες* → ανέβασε εικόνα → **Publish**
- Σε 1–2 λεπτά η φωτογραφία εμφανίζεται στη σελίδα (το Netlify ξανα-δημοσιεύει αυτόματα)

Μπορείς επίσης να αλλάξεις σειρά (drag & drop) ή να διαγράψεις φωτογραφίες.

  για γρήγορη φόρτωση.
